# Environment Setup
To reproduce the environment for this project, please install the following dependencies in order.

## 1. Install PyTorch (CUDA 12.1)

```bash
pip install torch==2.5.1 torchvision==0.20.1 torchaudio==2.5.1
```

The following packages will be installed:

```ini
torch==2.5.1
torchvision==0.20.1
torchaudio==2.5.1
numpy==2.2.5
sympy==1.13.1
```

## 2. Install Transformers

```bash
pip install transformers==4.57.1
```

The following packages will be installed:

```ini
transformers==4.57.1
tqdm==4.67.1
huggingface-hub==0.36.0
```

## 3. Install Jsonargparse

```bash
pip install jsonargparse==4.42.0
```

The following packages will be installed:

```ini
jsonargparse==4.42.0
```

## 4. Install Accelerate

```bash
pip install accelerate==1.11.0
```

The following packages will be installed:

```ini
accelerate==1.11.0
```

## 5. Install Ray

```bash
pip install ray==2.51.1
```

The following packages will be installed:

```ini
ray==2.51.1
```

## 6. Install Pandas

```bash

```

The following packages will be installed:

```ini

```

## 7. Install MTEB
